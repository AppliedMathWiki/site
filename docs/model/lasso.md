# LASSO

## Model

This is a test

$$
    \mathsf{\min_{x \in \mathbb{R}} \|Ax - b\|_2^2 + \lambda \|x\|_1. }
$$

## Overview

Classic scheme proposed by Tibshirani.

## Properties

Convex. Biased.

- Compressed sensing...
